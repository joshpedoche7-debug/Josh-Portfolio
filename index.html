Godzilla Bot

Godzilla Bot is a multi-purpose robotic project designed to showcase creativity, innovation, and the practical application of robotics. This bot was built with different functions that allow it to perform various tasks such as line following, sumo battles, hockey gameplay, and maze solving. It uses sensors, motors, and programmable controls to navigate environments, respond to obstacles, and complete specific challenges. The goal of this project is to explore how robotics can be used to solve problems while improving skills in programming, electronics, and system design. Godzilla Bot represents the combination of technology and creativity, showing how a single robotic platform can be adapted for different activities and competitions. Through this project, we were able to experience the process of building, testing, and improving a functional robot.
